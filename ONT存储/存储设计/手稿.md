## 交互协议

1. Request

   ```
   1.1 Get 请求原语
   1.2 Put 请求原语
   1.3 New 请求原语
   1.4 Free 请求原语
   1.5 Settlement 请求原语
   ```

2. Response

   ```
   2.1 Get 响应原语
   2.2 Put 响应原语
   2.3 New 响应原语
   2.4 Free 响应原语
   2.5 Settelment 响应原语
   ```

   ```
   1.1 Check 请求原语
   1.2 Check 响应原语
   ```
